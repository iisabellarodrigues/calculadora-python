# Calculadora em Python

## Descrição
Este projeto consiste em uma calculadora simples desenvolvida em Python, capaz de realizar operações matemáticas básicas por meio de uma interface de linha de comando.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de divisão por zero
- Validação de entradas do usuário

## Tecnologias Utilizadas
- Python 3
